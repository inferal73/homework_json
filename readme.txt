1. Установить Node.js
https://nodejs.org/en/

2. Установить git bash
https://gitforwindows.org/

3. Зайти в git bash и перейти в нем в папку проекта

4. Выполнить команду npm install

5. После выполнения npm install выполните команду node server.js

6. Терминал должен подвиснуть с сообщением о том что сервер запустился

7. Закрыв терминал или нажав ctrl+C вы остановите серве
